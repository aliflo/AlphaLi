Manual for AlphaLi

AlphaLi is a program for analysing data and visualising graphs. It focuses on two main datasets: global Malaria cases and google search trends for the term "Artificial Intelligence", but also supports user-uploaded data.

It was created as part of an EPQ project by Tom Birkbeck and Callum Cafferty.

1. Using the program

Once the program is launched, you can see four buttons along the left hand side.

  1. Canvas Manipulator - This button allows you to manipulate canvas scale, zoom and change pen colour as well as clearing the canvas. Clicking it will open a menu that displays these options.
  2. Equation - This button, when pressed, prompts the user to enter an equation which will then be graphed. This allows the user to compare a plotted dataset to a simple model.
  3. Data Analysis - Pressing this button brings up the Data Analysis menu, which allows the user to select the two inbuilt datasets or enter/upload their own and perform different kinds of regression on them.
  4. Clear Canvas - Clears any lines that have been drawn on the canvas
  5. Manual - Button to open this page in the default web browser. This page is an html file saved in the program folders and is located by the python program and launched through the default launcher.

4. Equation

When pressed, this button brings a prompt box for the user to type into. The user should enter the equation as a text string with no scientific notation and using only numbers and x.Equations can be written pythonically or in basic mathematical notation. Equations could look like so:

x^2+3x+5

x**5+10*x+3.4

x^6+9*x+5

(3x)e^2x
