Manual for AlphaLi

AlphaLi is a program for analysing data and visualising graphs. It focuses on two main datasets: global Malaria cases and google search trends for the term "Artificial Intelligence", but also supports user-uploaded data.

It was created as part of an EPQ project by Tom Birkbeck and Callum Cafferty.

1. Using the program

Once the program is launched, you can see four buttons along the left hand side.

  1. Canvas Manipulator - This button allows you to manipulate canvas scale, zoom and change pen colour as well as clearing the canvas. Clicking it will open a menu that displays these options.
  2. Data Analysis - Pressing this button brings up the Data Analysis menu, which allows the user to select the two inbuilt datasets or enter/upload their own and perform different kinds of regression on them.
  3. Manual - Button to open this page in the default web browser. This page is an html file saved in the program folders and is located by the python program and launched through the default launcher.
  4. Equation - This button, when pressed, prompts the user to enter an equation which will then be graphed. This allows the user to compare a plotted dataset to a simple model.
  5. Clear Canvas - Clears any lines that have been drawn on the canvas

Equation

When pressed, this button brings a prompt box for the user to type into. Polynomials are supported, of degree 1, 2 and 3. The user should enter the equation as a text string with no scientific notation and using only numbers and x. Valid examples include:

Valid Invalid
2x

3.4e-05x+35
2x^2-4 (2.1*10^4)x^3
0.05x^3-45x^2+213.2 2x+3x-sqrt(x)
